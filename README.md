# 这个分支
来源于ArHanXing的一次发电：
```md
空岛，开局只有一瓶药水和一根末地烛。玩家和猫娘有一个attribute，作为开发度，在计算末地烛威力的时候乘算；
末地烛威力越大，可以得到tier1-5纯度的欲望水晶，直接作为万物之源，并参与后续合成。
关于猫娘：你可以捕捉猫娘，像林业养蜂一样在机器里开发她（受限于技术，猫娘肯定会变成带有nbt的捕捉版猫娘）并取得欲望水晶；
猫娘的开发度也有上限并与教培欲望正相关，超限会死亡或者疯狂。
关于自己：开发度的增加肯定是有正面反馈，除了水晶纯度等级上升还有速度加快。
后期甚至可以做群星四飞升那样的强化路线（？）
```
